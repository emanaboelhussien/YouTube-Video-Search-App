YouTube Video Search App
الوصف (Description)

هذا التطبيق هو جزء من متطلبات الواجب الثاني (Assignment 2) لمساق تطوير تطبيقات الأندرويد. يتيح التطبيق للمستخدمين البحث عن فيديوهات يوتيوب عبر واجهة بسيطة، حيث يتم جلب البيانات برمجياً من YouTube API وعرضها بشكل منظم.
المميزات الرئيسية (Key Features)

    Search Functionality: إمكانية البحث عن أي موضوع والحصول على نتائج فورية.

    Background Processing: استخدام AsyncTask لتنفيذ عمليات الشبكة في الخلفية لضمان سلاسة التطبيق.

    Modern UI: عرض النتائج باستخدام RecyclerView لتقديم تجربة مستخدم متميزة.

    Image Loading: استخدام مكتبة Glide لتحميل الصور المصغرة للفيديوهات بكفاءة.

التقنيات المستخدمة (Technologies Used)

    Language: Java.

    API: YouTube Data API v3.

    Networking: HttpURLConnection لجلب بيانات JSON.

    JSON Parsing: تحليل الردود لاستخراج العناوين، الأوصاف، والروابط.

خطوات التشغيل (Setup and Installation)

    قم بعمل Clone لهذا المستودع على جهازك أو تحميله كـ Zip.

    افتح المشروع باستخدام Android Studio.

    تأكد من عمل Gradle Sync لتحميل المكتبات اللازمة.

    هام: يجب إضافة YouTube API Key في ملف NetworkUtils.java لكي يعمل البحث بشكل صحيح.

    قم بتشغيل التطبيق على محاكي (Emulator) أو هاتف حقيقي.
